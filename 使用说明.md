# 个人公开信息搜索系统

## 功能简介

本系统用于根据简历信息搜集个人公开信息，包括：

1. **全网搜索** - 搜索新闻、公众号等自媒体报道
2. **企业信息查询** - 查询企查查、爱企查、天眼查上的企业任职信息
3. **法律纠纷查询** - 查询中国裁判文书网、审判流程信息公开网等法律信息
4. **重名信息筛选** - 自动筛选和匹配，排除重名干扰

## 环境要求

- Python 3.8+
- 依赖库：见 requirements.txt

## 安装步骤

1. 确保已安装 Python 3.8+
2. 安装依赖库：
```bash
pip install -r requirements.txt
```

## 使用方法

### 方式一：运行 run.bat（推荐）

双击运行 `run.bat`，然后选择运行模式：
- 1: 交互式输入模式
- 2: 简历文件模式
- 3: 启动Web网站
- 4: 帮助信息

### 方式二：命令行模式

```bash
# 交互式输入
python main.py --cli

# 简历文件模式
python main.py "d:\简历\张三简历.pdf"

# Web网站模式
python app.py
```

### 方式三：Web网站（推荐）

1. 启动服务：
```bash
python app.py
```

2. 在浏览器中访问：http://localhost:5000

3. 上传简历文件或手动输入信息，点击搜索即可

## 支持的简历格式

- Word文档 (.docx)
- PDF文档 (.pdf)
- 图片文件 (.jpg, .jpeg, .png, .bmp) - 需要OCR支持
- 文本文件 (.txt)

## Web网站功能

### 首页功能
- 支持上传简历文件自动解析
- 支持手动输入姓名、企业、岗位等信息
- 拖拽上传支持

### 结果页功能
- 展示网络新闻搜索结果
- 展示企业信息查询结果
- 展示法律纠纷信息
- 相关度评分显示
- 报告下载（Excel、Word）
- 打印功能

## 输出格式

支持三种输出格式：
- Excel (.xlsx)
- Word (.docx)
- JSON (.json)

## 注意事项

1. 部分网站（如中国裁判文书网）需要登录才能访问详细信息
2. 请遵守各网站的使用条款和robots.txt
3. 建议适当控制搜索频率，避免被封禁IP
4. 本系统仅供合法合规用途使用
5. 图片OCR功能需要安装 Tesseract OCR 引擎

## 安装 Tesseract OCR（可选）

图片解析需要安装 Tesseract OCR：
- Windows：https://github.com/UB-Mannheim/tesseract/wiki
- macOS：brew install tesseract
- Linux：sudo apt-get install tesseract-ocr
